## Core Packages

Install the core packages to get started:

```bash
npm install @spoosh/core @spoosh/react
```

Or with other package managers:

```bash tab="pnpm"
pnpm add @spoosh/core @spoosh/react
```

```bash tab="yarn"
yarn add @spoosh/core @spoosh/react
```

```bash tab="bun"
bun add @spoosh/core @spoosh/react
```

## Plugins

Install only the plugins you need:

```bash
# Caching
npm install @spoosh/plugin-cache

# Automatic retry
npm install @spoosh/plugin-retry

# Cache invalidation after mutations
npm install @spoosh/plugin-invalidation

# Optimistic updates
npm install @spoosh/plugin-optimistic

# Polling
npm install @spoosh/plugin-polling

# Request debouncing
npm install @spoosh/plugin-debounce

# Request throttling
npm install @spoosh/plugin-throttle

# Request deduplication
npm install @spoosh/plugin-deduplication

# Prefetching
npm install @spoosh/plugin-prefetch

# Refetch on focus/reconnect
npm install @spoosh/plugin-refetch

# Initial data
npm install @spoosh/plugin-initial-data

# Next.js integration
npm install @spoosh/plugin-nextjs

# Debug logging
npm install @spoosh/plugin-debug
```

## Recommended Setup

For most projects, we recommend starting with these plugins:

```bash
npm install @spoosh/core @spoosh/react @spoosh/plugin-cache @spoosh/plugin-retry @spoosh/plugin-invalidation
```

This gives you:
- **Caching** - Avoid redundant network requests
- **Retry** - Automatically retry failed requests
- **Invalidation** - Auto-refresh related queries after mutations

## Framework Adapters

### Hono

If you're using Hono on the server, install the type adapter:

```bash
npm install @spoosh/hono
```

This lets you automatically infer your API schema from Hono routes.

### OpenAPI

To generate OpenAPI specs from your schema:

```bash
npm install @spoosh/openapi
```

## TypeScript Configuration

Spoosh requires TypeScript 5.0 or later. Ensure your `tsconfig.json` has strict mode enabled for the best type inference:

```json title="tsconfig.json"
{
  "compilerOptions": {
    "strict": true,
    "target": "ES2020",
    "module": "ESNext",
    "moduleResolution": "bundler"
  }
}
```
